# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 概要

このリポジトリはAWSのロギングと監査イベントについて学習するためのドキュメントリポジトリです。CloudTrailイベントのリファレンス資料を含んでおり、主にS3 APIアクションに焦点を当てています。

## リポジトリの内容

- **S3.md**: CloudTrailでサポートされているS3関連の監査イベントの包括的なリスト
  - Amazon S3標準アクション（GetObject、PutObject、DeleteObjectなど）
  - Amazon S3 Controlアクション（Access Grants、Access Points、Storage Lensなど）
  - Amazon S3 on Outpostsアクション
  - Amazon S3 Tablesアクション
  - Amazon S3 Vectorsアクション

## 目的

セキュリティ監査とコンプライアンスのため、CloudTrailロギングで監視可能なAWS APIアクションを理解するためのリファレンスとして使用します。
